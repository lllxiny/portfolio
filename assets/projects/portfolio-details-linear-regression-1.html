Linear Regression-1

This code demonstrates the use of various statistical techniques, including linear and logistic regression, bootstrapping, interaction analysis, and model evaluation.

1. Linear Regression for Sales Prediction: It compares models with and without interaction terms, estimates treatment effects, and calculates standard errors. 2. Logistic Regression for Purchase Prediction: The code builds models with and without interaction terms, interprets coefficients, and evaluates model accuracy. 3. Model Interpretation: The impact of predictor variables on outcomes is explained using odds ratios. 4. Model Evaluation: Model performance is assessed through accuracy scores, classification reports, and confusion matrices. 5. Interaction Plots: The code visualizes interactions between 'Age' and 'Gender' variables with the 'Purchased' outcome.
